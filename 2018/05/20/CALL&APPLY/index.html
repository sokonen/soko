call.apply.bind

call和apply可以改变this的指向
例

call()参数第一个为改变的this对象，后面的为函数的参数
apply()第一个参数于call()相同，后面以数组的形式向函数传参

引用其他对象的方法

apply()
可以用于push方法的数组连接
var arr1 = [1,2];
var arr2 = [3,4,5];
arr1.push.apply(arr1,arr2);

取最大值最小值
Math.max.apply(null,array);

将看起来像数组但不是数组的串转为数组
Array.prototype.slice.call(document.getElementsByTagName(“*”));
Array.prototype.slice.call(arguments);

bind
于call或者apply不同的是，bind并不会直接运行改变this指向之后的函数，
而是返回一个改变了this指向之后的函数。
