AJAX和JSONP

创建
new ActiveXObject(“microsoft.xmlhttp”);
new XNLHttpRequest();

监听
ajax.onreadystatechange
其中
——readyState 存放着XMLHttpRequest的状态
0：请求未初始化
1：服务器连接已经建立
2：请求已经接受
3：请求处理中
4：请求处理完毕，相应已经就绪
——status
200：“OK”
404：未找到页面
可见当readyState==4且status==200时表示响应就绪
即
···
if (xmlhttp.readyState==4 && xmlhttp.status==200)
{
/code/
}
···
服务端返回的数据——xmlhttp.responsetext
——ajax.open()
其中第一个参数为提交方式（post|get）
第二个参数为路径url
——ajax.send()
提交方式为get时参数为null
提交方式为post时用来传递参数
此时需要设定头部信息
ajax.setRequestHeder(“Content-Type”,”application/x-www-form-urlencoded;charset=UTF-8”);
ajax.send(“username=”+username+”&password=”+password);

JSONP
出于浏览器同源策略规定，ajax无法进行跨域请求，但是html标签中的src属性确允许进行跨域操作
利用这一特性通过script标签返回一个带参数的函数调用。
前端js实现
var src = ‘http://127.0.0.1:8080/jsonp?callback=test&namevl=abc'；
var script = document.createElement(‘script’);
script.setAttribute(‘type’, ‘text/javascript’);
script.src = src;
document.body.appendChild(script);
通过body中的脚本文件动态向body中添加script标签，并通过标签请求路径返回的内容调用本地全局函数。
node服务器
解析字符串后取得相应信息并返回需求的函数和参数。
使用querystring
var qs = querystring.parse(req.url.split(‘?’)[1]);
其中 qs.callback = test
qs.namevl = abc
var data = {
“name”: qs.namevl
};
data = JSON.stringify(data);
var callback = qs.callback + ‘(‘+data+‘);’;
res.end(callback);
